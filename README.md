🚖 **Call Taxi Booking Application**

A simple Java Console-Based Call Taxi Booking System developed using Object-Oriented Programming concepts and package structure.

📌 Project Structure

CallTaxiBookingApplication
│
├── src
│   ├── com.calltaxi.main
│   │      └── Main.java
│   │
│   ├── com.calltaxi.model
│   │      ├── Booking.java
│   │      └── Taxi.java
│   │
│   ├── com.calltaxi.service
│   │      └── TaxiService.java
│   │
│   └── module-info.java


**📦 Packages Explanation**

**🔹 com.calltaxi.main**

Contains Main.java

Entry point of the application.

Handles user input and menu options.

**🔹 com.calltaxi.model**
Contains:

Booking.java → Booking details (customer, pickup, drop, time, fare).

Taxi.java → Taxi details (Taxi ID, earnings, current location).

**🔹 com.calltaxi.service**

Contains:

TaxiService.java → Business logic for:

Taxi allocation

Fare calculation

Booking management

**⚙️ Features**

Add taxi details

Book a taxi

Allocate nearest available taxi

Calculate fare

Track taxi earnings

Display booking details
